# Twitter Extension

Chrome extension for enhanced Twitter/X functionality with content script injection and popup UI.

## Files

- `manifest.json` — Chrome Manifest V3 configuration
- `content.js` — Content script injected into Twitter pages
- `background.js` — Service worker for background processing
- `popup.html` / `popup.js` — Extension popup interface

## Installation

1. Open `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this directory
